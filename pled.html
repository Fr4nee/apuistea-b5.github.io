Introduccion

En la materia Programacion Logica y en Estructura de Datos usamos el lenguaje C#, este es la evolcion de C++ (C++++), es un lenguaje multiparadigma desarrollado por Microsoft e implementado en su plataforma .NET. Su sintaxis básica deriva de C/C++ y utiliza el modelo de objetos de la plataforma .NET, similar al de Java, aunque incluye mejoras derivadas de otros lenguajes.

Programacion C# en VSC
Link de descarga Visual Studio Code

Hay que tener en cuenta que no es lo ideal programar con C# en Visual Studio Code ya que no tiene algunas caracteristicas fundamentales pero debido a que a muchos no les gusta o no prefieren Visual Studio 2019 por cuestiones de rendimiento es posible programar aplicaciones de consola.

Para poder crear proyectos de C# en VSC hay que contar con la extension de C# y el framework SDK de .NET 5.0 o posterior

  • 1) Creamos una carpeta del proyecto.
  • 2) Abrimos la carpeta en VSC
  • 3) Iniciamos un terminal, preferentemente en cmd.
  • 4) Escribimos dotnet new console en la terminal. Esto va a generar los archivos necesarios para empezar a programar.
  • 5) Para ejecutar el codigo escribimos dotnet run en la terminal.

Crear Proyecto en Visual Studio: Aplicacion de Consola.

  • 1) En la seccion de Tareas Iniciales seleccionamos Crear proyecto.
  • 2) En Plantillas buscamos Aplicacion de Consola.
  • 3) Escribimos el nombre del proyecto, el nombre de la solucion (deberian ser iguales) y definimos la ruta.
  • 4) En Plataforma de Destino elegimos .NET 3.1 Compatibilidad a largo plazo para tener la version mas nueva y estable.
  • 5) Crear.

Entry Point (Programa Basico)

Por defecto, cuando creamos un proyecto en Visual Studio, se genera un Entry Point que es el metodo donde inicia nuestro programa. A continuacion se mostrara como es en el IDE y el resultado en consola.

Codigo en IDE.

Resultado en Consola.

En la imagen del IDE podemos ver que el Entry Point Static void Main(string[] args) {...} esta conformado por los siguientes elementos:

static: Es un modificador que permite ejecutar un método sin tener que instanciar a una variable (sin crear un objeto). El método Main() debe ser estático.

void Indica el tipo de valor de retorno del método Main(). No necesariamente tiene que ser void.

string[] args Es un Array de tipo string que puede recibir el método Main() como parámetro.

Ademas en la imagen del IDE podemos observar otros elementos importantes:

public static class Program: Representa la entrada, salida y errores de Streams para aplicaciones de consola. Es miembro del NameSpace ConsoleApp1
